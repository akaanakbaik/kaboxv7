# domku box

CDN Upload Service dengan 8 Provider Random

Fitur:
- Drag & Drop max 5 file
- Progress bar persen real-time
- 8 Provider Random: Supabase, Neon, Turso, Appwrite, Cloudinary, Backblaze, ImageKit, Prisma
- Expired: 30 detik / 1 menit / 1 hari / 30 hari / 1 tahun / Selamanya
- Auto IP bahasa (ID/EN)
- API public tanpa key
- Telegram monitoring + bot kontrol
- Full security

Deploy ke Vercel:
1. Upload semua file ke GitHub
2. Vercel → New Project → Import
3. Tambahkan semua variabel dari .env.example
4. Deploy
