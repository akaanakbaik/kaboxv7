# domku box

CDN Upload Service Super Cepat & Aman

Fitur Lengkap:
- Drag & Drop multi file (max 5)
- Progress bar persen real-time
- Multi Storage Random (semua database digunakan secara merata)
- Pilih masa expired: 30 detik / 1 menit / 1 hari / 30 hari / 1 tahun / Selamanya
- Auto redirect bahasa berdasarkan IP (Indonesia = /id/ | luar = /en/)
- API public tanpa key
- Rate limit ketat 10 req/detik + auto block IP
- Telegram monitoring real-time + bot kontrol 20+ fitur
- Security full: Helmet CSP, no XSS, no SQLi, no leak env
- Mobile + Desktop perfect
- Full animasi smooth, no gradient, modern clean

Cara Deploy ke Vercel:
1. Buat repo GitHub baru
2. Upload semua file ini persis sesuai folder
3. Vercel → New Project → Import repo
4. Tambahkan Environment Variables dari .env.example ke Vercel
5. Deploy (otomatis jalan)

Custom Domain:
Settings → Domains → tambah domku.xyz
