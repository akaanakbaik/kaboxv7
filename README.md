# domku box

CDN Upload Service dengan 8 Provider Random

Fitur Lengkap:
• Drag & Drop max 5 file
• Progress bar persen real-time
• 8 Provider Random: Supabase, Neon, Turso, Appwrite, Cloudinary, Backblaze, ImageKit, Prisma
• Expired: 30 detik / 1 menit / 1 hari / 30 hari / 1 tahun / Selamanya
• Auto IP bahasa (Indonesia = /id/ | luar = /en/)
• API public tanpa key
• Telegram monitoring + bot kontrol
• Keamanan tinggi (Helmet CSP, rate limit 10 req/detik, IP block)
• Full animasi smooth, mobile & desktop friendly

Semua 8 database/provider wajib terpakai secara acak setiap upload.

Cara Deploy:
1. Buat repo GitHub baru
2. Upload semua file (struktur persis)
3. Vercel → New Project → Import repo
4. Tambahkan SEMUA Environment Variables dari .env.example
5. Deploy
