domku box - CDN Upload Service

Fitur Lengkap:
- Drag & drop + multi file (max 5)
- Progress bar persen real-time
- Auto bahasa (ID/EN) berdasarkan IP
- API public tanpa key
- Security rate limit 10 req/detik + blokir
- Telegram monitoring otomatis
- Mobile & desktop friendly
- Full animasi smooth, no gradient

Cara Deploy ke Vercel:
1. Buat repo baru di GitHub
2. Upload semua file ini
3. Buka vercel.com → New Project → Import repo
4. Deploy (Vercel otomatis detect Vite + Node)
5. Set custom domain di Vercel Settings → Domains → tambah domku.xyz
